# Proyecto Intermodular - DUOFUT

Este proyecto consiste en el diseño de una aplicación llamada **DUOFUT**.

DUOFUT es una app inspirada en Duolingo, pero enfocada al aprendizaje del fútbol. El usuario podrá aprender diferentes aspectos del fútbol como jugador, entrenador, árbitro o aficionado.

El objetivo del proyecto no es programar la aplicación, sino organizar, planificar y documentar todo el proceso como si fuera un proyecto real de desarrollo.
