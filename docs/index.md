# DUOFUT

Bienvenido a la documentación del proyecto **DUOFUT**.

DUOFUT es una aplicación de aprendizaje basada en el funcionamiento de Duolingo, pero centrada en el fútbol.

El usuario podrá aprender de forma progresiva diferentes áreas del fútbol como:

* Jugador
* Entrenador
* Árbitro
* Aficionado

## Objetivo del proyecto

El objetivo es diseñar una aplicación completa, organizando el trabajo como lo haría un equipo profesional.

En esta documentación se recoge todo el proceso del proyecto:

* Organización del equipo
* Definición de la idea
* Planificación
* Diseño de la aplicación

La documentación se irá ampliando a medida que avance el proyecto.
