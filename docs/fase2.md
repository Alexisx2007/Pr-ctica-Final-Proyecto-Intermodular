# Fase 2 - Definición del proyecto

## Idea del proyecto

La aplicación que voy a diseñar se llama **DUOFUT**.

Es una app parecida a Duolingo, pero en lugar de enseñar idiomas, enseña fútbol.

---

## Objetivo

El objetivo es que cualquier persona pueda aprender fútbol de forma sencilla, interactiva y progresiva.

---

## Público objetivo

La aplicación está dirigida a:

* Jugadores
* Entrenadores
* Árbitros
* Aficionados al fútbol

---

## Problema que resuelve

Actualmente no existe una aplicación clara para aprender fútbol desde diferentes puntos de vista.

Con DUOFUT se pretende:

* Aprender desde cero
* Mejorar conocimientos
* Tener todo el contenido organizado

---

## Funcionamiento

El usuario podrá elegir qué tipo de aprendizaje quiere:

* Jugador
* Entrenador
* Árbitro
* Aficionado

La app tendrá niveles y lecciones, siguiendo un sistema similar a Duolingo.
