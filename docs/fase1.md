# Fase 1 - Creación del equipo y entorno de trabajo

## Canal de comunicación

Para organizarnos durante el proyecto, he creado un equipo en Microsoft Teams.

He utilizado una cuenta personal para evitar restricciones y poder trabajar con todas las funcionalidades disponibles. Este canal lo usamos para comunicarnos, compartir archivos y organizarnos.

---

## Organización de carpetas

He creado la estructura de carpetas del proyecto para mantener todo organizado.

Las carpetas son:

* docs/
* recursos/
* gantt/
* trello/
* diseño/
* README.md

Esto permite separar cada parte del proyecto y trabajar de forma más ordenada.

---

## Documentación en Markdown y MkDocs

He creado la carpeta **docs/** donde irá toda la documentación del proyecto.

Dentro he creado los archivos iniciales:

* index.md
* fase1.md
* fase2.md

Después he configurado MkDocs para poder visualizar la documentación como una página web.

Uso Markdown porque es fácil de escribir y MkDocs porque permite organizar la información y navegar entre secciones de forma clara.

También he subido el proyecto a GitHub para poder trabajar junto a mi compañero y mantener todo actualizado.
